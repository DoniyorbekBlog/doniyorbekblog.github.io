Basic HTML Links

HTML links are created using the <a> tag:

<a href="https://example.com">Visit Example</a>

Example link:

Visit Example Website

Link Colors and States

Different link states can be styled with CSS:

  • a:link – normal, unvisited link
  • a:visited – link already visited
  • a:hover – link when mouse is over it
  • a:active – link while being clicked
a:link {
  color: green;
  text-decoration: none;
}

a:visited {
  color: pink;
}

a:hover {
  color: red;
  text-decoration: underline;
}

a:active {
  color: yellow;
  text-decoration: underline;
}
