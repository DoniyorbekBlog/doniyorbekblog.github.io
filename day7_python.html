Python Lesson: filter()

filter() allows you to keep only the elements that satisfy a condition, without writing loops.

1. Basic Structure

filter(function, iterable)

It returns only the items for which function(item) is True.

You usually wrap it in list() to see the result:

list(filter(...))

2. Example: Keep Even Numbers

Using filter():


numbers = [1, 2, 3, 4, 5, 6]
def is_even(n):
    return n % 2 == 0
evens = list(filter(is_even, numbers))
print(evens)   # [2, 4, 6]

3. Using filter() With lambda (Most Common Way)

A lambda is a short, inline function.


numbers = [10, 21, 32, 43, 54]
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)  # [10, 32, 54]

4. Filtering Strings

Keep only long words (length > 3):


words = ["api", "python", "ai", "data", "go"]
long_words = list(filter(lambda w: len(w) > 3, words))
print(long_words)   # ['python', 'data']

5. Filtering out empty values

Very common in data cleaning.


items = ["hello", "", None, "world", 0, " ", "python"]
cleaned = list(filter(None, items))
print(cleaned)  
# ['hello', 'world', ' ', 'python']

Explanation:

filter(None, iterable) removes values that are:

  • empty string ""
  • o
  • None
  • False
  • empty lists/dicts/tuples
