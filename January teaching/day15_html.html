Learning HTML

Topic: HTML Images

Introduction

Images improve the design and appearance of a web page. The <img> element allows you to embed images directly into HTML.

Basic Examples

<img src="pic_trulli.jpg" alt="Italian Trulli">
    
<img src="img_girl.jpg" alt="Girl in a jacket">
    
<img src="img_chania.jpg" alt="Flowers in Chania">

HTML Images Syntax

The <img> tag creates a space for an image. Images are not stored inside the page; they are linked.

The <img> tag has no closing tag and contains attributes only.

Required attributes:

  • src – the image path (URL)
  • alt – text shown if the image cannot load
Syntax:
<img src="url" alt="alternatetext">

The src Attribute

The browser retrieves the image from the URL when the page loads. Make sure the file exists; otherwise the browser shows a broken image symbol and the alt text. 

<img src="img_chania.jpg" alt="Flowers in Chania">

The alt Attribute

The alt attribute describes the image. It is used when the image is missing or when screen readers read the page. 

<img src="wrongname.gif" alt="Flowers in Chania">

Screen readers read the alt text aloud for visually impaired users.

Image Size – Width and Height

You can set size using style:

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

Or using attributes:

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Always specify dimensions to avoid page flickering during load.

Width/Height vs. Style Attribute

All three are valid: width, height, and style. Using style ensures CSS files won’t accidentally override dimensions. 

<!DOCTYPE html>
<html>
<head>
<style>
img {
  width: 100%;
}
</style>
</head>
<body>

<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

</body>
</html>

Images in Another Folder

<img src="/images/html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

Images on Another Server

<img src="https://www.w3schools.com/images/w3schools_green.jpg" 
     alt="W3Schools.com">

External images may be copyrighted or changed without notice.

Animated GIFs

<img src="programming.gif" alt="Computer Man" style="width:48px;height:48px;">

Using an Image as a Link

<a href="default.asp">
  <img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

Image Floating

<p><img src="smiley.gif" alt="Smiley face"
style="float:right;width:42px;height:42px;">
The image will float to the right of the text.</p>

<p><img src="smiley.gif" alt="Smiley face"
style="float:left;width:42px;height:42px;">
The image will float to the left of the text.</p>

Chapter Summary

  • Use <img> to display images
  • Use src for the image URL
  • Use alt for alternate text
  • Use width, height, or style to control size
  • Use float to position images left or right
  • Large images slow down page performance
