Python sorted() Function — Complete Guide

1. What is sorted()?

The sorted() function returns a new list with items in sorted order. It does not change the original object.

numbers = [4, 1, 6, 2] result = sorted(numbers) print(result) # [1, 2, 4, 6] print(numbers) # Original list stays unchanged

2. Syntax

sorted(iterable, *, key=None, reverse=False)

Parameters:

  • iterable — list, tuple, string, dictionary keys, etc.
  • key — function to customize sorting (very powerful)
  • reverse — sort descending when True

3. Sorting Strings

print(sorted("python")) # ['h', 'n', 'o', 'p', 't', 'y']

4. reverse=True (Descending Order)

nums = [3, 9, 1, 5] print(sorted(nums, reverse=True)) # [9, 5, 3, 1]

5. key= — Custom Sorting Rules

You can sort using your own logic.

Sort by length of each word:

words = ["apple", "hi", "banana", "yes"] print(sorted(words, key=len)) # ['hi', 'yes', 'apple', 'banana']

Sort ignoring case:

names = ["Bob", "alice", "Steve", "john"] print(sorted(names, key=str.lower)) # ['alice', 'Bob', 'john', 'Steve']

6. Sorting Complex Objects

Sort a list of dictionaries by age:

people = [ {"name": "Ali", "age": 19}, {"name": "John", "age": 25}, {"name": "Tim", "age": 17} ] sorted_people = sorted(people, key=lambda p: p["age"]) print(sorted_people)

7. Sorting Tuples

Sort by second element:

pairs = [(2, 5), (1, 9), (4, 3)] sorted_pairs = sorted(pairs, key=lambda x: x[1]) print(sorted_pairs) # [(4, 3), (2, 5), (1, 9)]

8. Sorting a Dictionary

Sort dictionary keys:

d = {"b": 2, "c": 1, "a": 3} print(sorted(d)) # ['a', 'b', 'c']

Sort by dictionary values:

print(sorted(d, key=lambda k: d[k])) # ['c', 'b', 'a']

9. Sorting Numbers Stored as Strings

data = ["1", "100", "12", "5"] print(sorted(data)) # ['1', '100', '12', '5'] (string order) print(sorted(data, key=int)) # ['1', '5', '12', '100'] (numeric order)
