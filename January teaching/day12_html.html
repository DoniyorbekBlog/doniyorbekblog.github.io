Today's Topic: HTML Links

What Are HTML Links?

HTML links allow users to click and navigate to other web pages, documents, sections of the same page, emails, phone numbers, external websites, and more. Links are created using the <a> (anchor) tag.

Basic Link Example

This is the simplest form of an HTML link:

<a href="https://www.google.com">Visit Google</a>

Opening a Link in a New Tab

If you want the link to open in a new browser tab, use the target="_blank" attribute: 

<a href="https://www.youtube.com" target="_blank">
    Open YouTube in a New Tab
</a>

Linking to a Section Inside the Same Page (Internal Link)

You can link to specific sections using IDs: 

<a href="#section2">Go to Section 2</a>

...

<h2 id="section2">This is Section 2</h2>

Email Links

Use mailto: to open the user's email app: 

<a href="mailto:example@example.com">
    Send Email
</a>
