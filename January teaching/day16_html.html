Step 5: The loading Attribute

Improve your page speed by controlling when images are loaded

Why it's useful

  • Improves page loading speed
  • Saves bandwidth by loading images only when needed
  • Very helpful for long pages with many images

Syntax

<img src="cat.jpg" alt="A cute cat" loading="lazy">

Values for loading

  • lazy → Load only when the image comes into view (scroll to it)
  • eager → Load immediately, even if not visible yet (default)
  • auto → Let the browser decide

Example Gallery

<h1>Gallery</h1>
<img src="cat.jpg" alt="A cute cat" loading="lazy">
<img src="dog.jpg" alt="A happy dog" loading="lazy">
<img src="bird.jpg" alt="A colorful bird" loading="lazy">

Images far down the page won’t load until you scroll to them. Makes your website faster and smoother.

Tip

  • Use loading="lazy" for all images below the fold
  • Use loading="eager" for hero images at the top of the page

Exercise
