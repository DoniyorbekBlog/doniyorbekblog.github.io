Task:

We need a function that can transform a number (integer) into a string.
What ways of achieving this do you know?
Examples (input --> output): 


    123  --> "123"
    999  --> "999"
    -100 --> "-100"

My code:

def number_to_string(num):
    num = num
    return str(num)
I have num = num part because in test cases there was tricks: if input is 1+2 my program must return "3", not "1+2". That's why I have done this. But...

Best Solution

def number_to_string(num):
    return str(num)
Yes it true, because there is something which I did not see.
It is str(num) here num automatically calculates tasks like 1+3, 3-1...
