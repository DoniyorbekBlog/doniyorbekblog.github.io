Problem: 
Return the number (count) of vowels in the given string.
We will consider a, e, i, o, u as vowels for this Kata (but not y).
The input string will only consist of lower case letters and/or spaces.

And my solution at the beginning:


target = "aeiou"
def counter(string):
    counter = 0
    for letter in string:
        if letter in target:
            counter += 1
    return counter

Actually it is corrected, but if i continue writing codes like this, consequence will not be good!
That's why I started regenerating my codes so that it must be professional

My best solution: 
def counter(string):
    return len([letter for letter in string if letter in "aeiou"])
