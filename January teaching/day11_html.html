HTML Styles – CSS

Understanding CSS Basics With Examples

CSS = Cascading Style Sheets

CSS stands for Cascading Style Sheets. It saves a lot of work and allows you to control the layout and appearance of multiple web pages all at once.

CSS = Styles, Colors, Text Manipulation, Layout, Boxes

What is CSS?

Cascading Style Sheets (CSS) is used to format the layout of a webpage. With CSS, you can control:

  • Colors, fonts, text size
  • Spacing between elements
  • Positioning and layout
  • Background images and colors
  • Different designs for different screens
  • And much more

Tip: “Cascading” means that styles applied to a parent element also apply to its children, unless a more specific style overrides it.

Using CSS

CSS can be added to HTML in three ways:

  1. Inline CSS – using the style attribute in HTML elements
  2. Internal CSS – using a <style> element inside <head>
  3. External CSS – linking to a separate .css file

External CSS is the most common method in real projects, but tutorials often use inline or internal CSS for learning.

Inline CSS

An inline CSS applies a style directly to an HTML element using the style attribute.


<h1 style="color:blue;">A Blue Heading</h1>

<p style="color:red;">A red paragraph.</p>

Internal CSS

Internal CSS is defined inside the <head> section using a <style> element.


<!DOCTYPE html>
<html>
<head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

External CSS

An external CSS file defines styles for multiple web pages.


<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

The external file must be saved with a .css extension and contain no HTML.


/* styles.css */
body {
  background-color: powderblue;
}
h1 {
  color: blue;
}
p {
  color: red;
}

Tip: With an external CSS file, you can change the design of an entire website by editing only one file.

CSS Colors, Fonts, and Sizes

Commonly used CSS properties include:

  • color – text color
  • font-family – font type
  • font-size – text size

<style>
h1 {
  color: blue;
  font-family: verdana;
  font-size: 300%;
}
p {
  color: red;
  font-family: courier;
  font-size: 160%;
}
</style>

CSS Border

The border property defines a border around an element.


p {
  border: 2px solid powderblue;
}

CSS Padding

Padding adds space between text and the border.


p {
  border: 2px solid powderblue;
  padding: 30px;
}

CSS Margin

Margin adds space outside the border.


p {
  border: 2px solid powderblue;
  margin: 50px;
}

Linking to External CSS

External files can be linked using absolute or relative paths:


<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">
    

<link rel="stylesheet" href="/html/styles.css">
    

<link rel="stylesheet" href="styles.css">

Chapter Summary

  • Use the style attribute for inline styling
  • Use the <style> element for internal CSS
  • Use the <link> element for external CSS
  • Use the <head> element to store style information
  • Use color for text colors
  • Use font-family for fonts
  • Use font-size for text sizes
  • Use border for borders
  • Use padding for spacing inside the border
  • Use margin for spacing outside the border
